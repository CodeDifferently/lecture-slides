## Lecture Overview
* Wordpress looks for template files in the following order:
    1. Page Template
    2. `page-{slug}.php`
    3. `page-{id}.php`
    4. `page.php`
    5. `singular.php`
    6. `index.php`

### Page Template
* If the page has a custom tempalte assigned, WordPress uses that file.
